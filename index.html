Size Model

Hours are computed from each tree's own dimensions, not size buckets. Two levers below control how steeply cost scales.

1.0 = cost rises in a straight line with canopy width. 1.2 (default) = a 50 ft canopy costs ~6.9× a 10 ft one, because you work an area, not a line.

You climb and rig a tree once. The largest service on a tree bills full hours; each additional service on that same tree bills at this fraction. 0.55 = 55%.

Applied to matching trees beyond the first in a quantity group. 0.9 = 10% faster on repeats. Set 1.0 to disable.

Risk & Access Multipliers

These stack additively, not multiplicatively. Live oak (1.25) + over a roof (1.35) + climb (1.30) = 1 + .25 + .35 + .30 = 1.90×, not 2.19×.
